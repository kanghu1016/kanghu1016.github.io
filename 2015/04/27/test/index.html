Sublime好用熱鍵整理

發表於 2015-04-27   |   分類於 開發工具   |  

工具與好用功能整理

SublimeText

編輯

Ctrl + X 刪除行

Ctrl + Enter 插入下一行

Ctrl + Shift + Enter 插入前一行

Ctrl + Shift + ↑ 往上移動一行

Ctrl + Shift + ↓ 往下移動一行

Ctrl + 左 或 右
讓你的游標可以快速的回到該行的最前面或是最後面。

shift + 左 或 右
每按一次會選擇一個字元，可以更加精準的選取自己要的部份。

Ctrl + shift + 左 或 右
從游標所在處，往前選取或者往後選取該行到底。

Ctrl + L 選取整行
選取游標在內的一行，連按l的話會往下選取下面的行數。

Ctrl + D 選取單字（可重複選取相同的）

Alt + F3 選取所有已選取的單字

Ctrl + /註解與移除註解

Ctrl + Shift + /區域註解

Ctrl + Shift + V貼上並符合縮排

Ctrl + K + num 收合第幾層的code 0打開

Ctrl + M 跳到最近的括弧

Ctrl + Shift + M 選取括弧內的所有文字

Ctrl + K + K 刪除游標後方所有文字

Ctrl + K + Backspace 刪除游標前方所有文字

Ctrl + Shift + D複製行

Ctrl + J與下一行結合為同一行

Ctrl + Y回覆上一步，沒有上一步時重複上個動作

Ctrl + U軟回覆(包括選取動作)

切割視窗

　
Alt + Shift + (2, 3, 4)垂直分割(2, 3, 4)個視窗

Alt + Shift + 1回覆到預設

Alt + Shift + 5水平垂直分割

Alt + Shift + (8, 9)水平分割(2, 3)個視窗

書籤　

Ctrl + F2切換書籤

F2到下一個書籤

Shift + F2到上一個書籤

Ctrl + Shift + F2清除書籤

文字操作　

Ctrl + K + U轉換為大寫

Ctrl + K + L轉換為小寫

導覽/Goto Anything　

Ctrl + G快速到達某一行Ctrl + ;快速到達變數通用　

Ctrl + K + B切換側欄

Ctrl + P快速開啟檔案

Ctrl + R快速到達符號(function, class, etc…)

Ctrl + Shift + P命令提示符(包含所有命令)

Ctrl + Shift + Alt + P切換狀態列搜尋/取代　

Ctrl + F搜尋

Ctrl + H取代

Ctrl + Shift + F檔案間搜尋分頁　

Ctrl + Shift + T還原最後關閉的分頁

Ctrl + PgUp換到上一個分頁

Ctrl + PgDown換到下一個分頁

Ctrl + Tab切換上一個分頁

Alt + [NUM]切換到第[NUM]個分頁

SublimeText Plugin Emmet

syntax

Child: >

Sibling: +

Climb-up: ^

Grouping: ()

div>(header>ul>li*2>a)+footer>p^section

compile

<div>
    <header>
        <ul>
            <li><a href=""></a></li>
            <li><a href=""></a></li>
        </ul>
    </header>
    <footer>
        <p></p>
    </footer>
    <section></section>
 </div>

Custom attributes: [a=’value1’]

Multiplication: *

Item numbering: $

h$[title=item$]{Header $}*3

compile

<h1 title="item1">Header 1</h1>
<h2 title="item2">Header 2</h2>
<h3 title="item3">Header 3</h3>

ID and CLASS attributes : .#

lorem
利用lorem加數字產生dummy paragraph

lorem100經轉換產生一段100個字的文章
